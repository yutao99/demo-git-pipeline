# \# demo-git-pipeline

# 

# A minimal repo to validate a clean Git workflow on Windows (Git Bash + SSH).

# 

# \## Goals

# \- Verify Git installation and basic commands

# \- Configure SSH authentication for GitHub

# \- Ensure commit author identity uses GitHub `noreply`

# \- Push to a clean remote repo with `main` tracking `origin/main`

# 

# \## Repo layout

# \- `src/`   : sample source code

# \- `docs/`  : documentation (kept with `.gitkeep`)

# \- `tools/` : helper scripts (kept with `.gitkeep`)

#

# ```bash

# git clone git@github.com:yutao99/demo-git-pipeline.git

# cd demo-git-pipeline

# git log --oneline --decorate -n 5
